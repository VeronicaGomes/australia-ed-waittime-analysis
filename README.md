# Australia's Public Hospital Wait Time Crisis
### A Business Analysis Report: Victoria vs New South Wales

## Overview
This project analyses the causes of declining Emergency Department (ED) performance across Victoria and New South Wales (NSW) using real government data. It was completed as an independent BA portfolio project to demonstrate end-to-end business analysis skills.

## Problem Statement
No Australian state met the national 72% NEAT benchmark in 2022-23. Victoria's 4-hour rule performance dropped from 74% to 56% over the past decade, while NSW declined from 88% to 74%, despite consistent increases in government funding.

## Project Deliverables
- Stakeholder Analysis and Power/Interest Matrix
- Data Analysis using AIHW Emergency Department Care Data (2011-12 to 2024-25)
- AS-IS Process Map of current ED patient journey
- Root Cause Analysis using Fishbone Diagram
- TO-BE Process Map with recommended improvements
- Power BI Dashboard with KPI cards, gauge charts, bar chart, scatter plot and trend line charts
- Recommendations and RACI Matrix
- Full Business Analysis Report
